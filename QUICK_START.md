## 🎮 FPV Simulator Use Case

Place your Android device in a VR box headset and:
1. Run the Android app
2. Connect to desktop server
3. Desktop screen mirrors in stereo VR format
4. Play drone FPV simulators with screen mirrored
5. Touch the screen to control (maps to desktop input)
