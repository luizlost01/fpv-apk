║                        TROUBLESHOOTING                             ║
╚════════════════════════════════════════════════════════════════════╝

❌ "Cannot connect" error:
   → Check server is running (look at terminal)
   → Phone on same WiFi as computer? (Settings → WiFi)
   → Firewall allowing ports 5555-5556? (sudo ufw allow 5555/tcp)

❌ "Connected but no video":
   → Restart both server and app
   → Check server terminal for errors
   → Try USB connection: adb forward tcp:5555 tcp:5555

❌ "App crashes on startup":
   → Check Android version is 5.0+ (Settings → About)
   → Check phone has 100MB+ free space
   → Grant WiFi permissions if prompted

❌ "Touch not working":
   → Check port 5556 open: sudo ufw allow 5556/tcp
   → Restart app and server
   → Check server terminal shows "Client connected"

# Find your computer IP
ifconfig | grep inet | grep -v 127

# Build and install APK
cd ~/fpv-screen-mirror/android && ./build-apk.sh

# Start server
cd ~/fpv-screen-mirror/desktop && python3 main.py

# View logs
adb logcat | grep fpv

# Install APK manually
adb install -r app/build/outputs/apk/debug/app-debug.apk

# Launch app
adb shell am start -n com.fpvmirror/.MainActivity

# USB port forwarding (if WiFi doesn't work)
adb forward tcp:5555 tcp:5555
adb forward tcp:5556 tcp:5556
